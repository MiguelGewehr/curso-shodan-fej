# Simple React Login Page

A clean and modern login page built with React and Vite. This project provides a simple and reusable login component with a user-friendly interface.

## Features

- Email and password input fields.
- "Remember me" checkbox and "Forgot password?" link.
- A link to register for a new account.
- User input is managed using React state.
- Icons from the React Icons library for better UX.

## Technologies Used

- **React 19**
- **Vite**
- **react-icons**
